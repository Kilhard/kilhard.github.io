Grid Gallery

Адаптивный блок, верстка Pixelperfect, применение grid технологии

Техническое задание от IT-компании "Открытые решения". Необходимо было сверстать адаптивный блок по 3 макетам. Требования к верстке следующие: html5, less/css (less-файлы не более 2-3 уровней вложенности), название классов согласно БЭМ, использование спрайтов. Минимальная ширина 320px, максимальная 768px фиксированная по центру родителя, внутри верстка "резиновая". Соотвествие макетам pixelperfect в брейкпоинтах 320, 480 и 768px.

Задание было успешно выполнено. Самое сложное - галерея, оформленная в стиле кирпичной мозаики. Нельзя было применять JS плагины и делать дополнительные div-обертки. Также нельзя было применять колоночный вариант, т.к. нужно было, чтобы фотографии следовали друг за другом слева направо. Поэтому я выбрал метод CSS через Grid

  • Дата: Июль 2018
  • Категории: html, css, pixelperfect
